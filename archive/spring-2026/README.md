# Spring 2026 Archive

These CSV files are offline snapshots used by the app as the primary source for Spring 2026.

## Files
- `oom.csv`
- `tqe-1.csv`
- `tqe-2.csv`
- `tqe-3.csv`

## Fallback env vars (Vercel)
If a local file is missing or fails to load, the app falls back to these env vars:
- `NEXT_PUBLIC_CSV_2026SPRING_OOM`
- `NEXT_PUBLIC_CSV_2026SPRING_TQE1`
- `NEXT_PUBLIC_CSV_2026SPRING_TQE2`
- `NEXT_PUBLIC_CSV_2026SPRING_TQE3`
